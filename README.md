# GitHub Connector Master Skill

A comprehensive, portable master skill for advanced usage of the GitHub connector.

**This is a skill, not an MCP server.** Load this after you already have a working GitHub connector/MCP installed.

It works across Grok (Cloud, iOS, Web, CLI), Claude, and other platforms that support loading `.md` skill files.

## What's Inside
- Full encyclopedia-style breakdown of all advanced GitHub connector capabilities
- Core operating principles (Connector-First, Timestamp Discipline, Private-First Development)
- Quick reference cheat sheets
- Ready-to-use templates for common tasks
- Anti-patterns to avoid
- Best practices

## Private-First Development Rule
All major work should start in a private development repository. Only publish or mirror to public after review and completion.

## Quick Start
1. Install/load a GitHub connector/MCP in your agent
2. Load `GitHub-Connector-Master-Skill.md` for full understanding
3. Use the cheat sheets and templates as needed
